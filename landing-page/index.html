SiteMorph

Community Edition

Modify any website using natural language. Powered locally on your device by Gemini Nano.

Download Extension (.zip)

How to Install

  1. Extract the downloaded sitemorph-extension.zip file.
  2. Open Google Chrome and navigate to chrome://extensions/
  3. Toggle Developer mode ON (top right corner).
  4. Click Load unpacked and select the extracted folder.
Note: You must have Chrome 140+ with the Prompt API enabled via chrome://flags for the local AI to work. No data leaves your browser.
