# SiteMorph (Community Edition)

SiteMorph is a Manifest V3 Chrome Extension that uses the built-in Gemini Nano (Prompt API) to manipulate website DOM/behavior via natural language.

## Prerequisites

SiteMorph uses the experimental local AI capabilities of Chrome. You must use Chrome version 140+ with specific flags enabled:

1. Go to `chrome://flags`
2. Search for and enable `#prompt-api-for-gemini-nano`
3. Search for and enable `#optimization-guide-on-device-model`
4. Restart Chrome

## Installation

```bash
npm install
npm run build
```

The extension will be built into the `/dist` directory.

## Loading the Extension

1. Go to `chrome://extensions/`
2. Enable "Developer mode" in the top right.
3. Click "Load unpacked" and select the `dist` folder generated by the build process.

## Using SiteMorph

1. Click the SiteMorph extension icon to open the side panel.
2. Enter a natural language request like "Make the background red" or use one of the built-in templates.
3. The extension will generate JavaScript using Gemini Nano and execute it on the active tab.
